# Artify — Creative Artwork Showcase

Live site: https://artify-client-d88f7.web.app/

Artify is a modern art-sharing platform where artists can upload, display,
and share their artworks. Key features include:

- Upload and manage artworks with images, descriptions, and categories.
- Explore public artworks with search and category filters.
- Like artworks and add them to your favorites.
- Private pages for adding artworks, user gallery, and favorites (Firebase Auth).
- Responsive, clean UI with light/dark theme toggling and animations.

---

## ✨ Features

- 🖼️ Upload and manage artworks with images, descriptions, and categories
- 🔍 Explore public artworks with search and category filters
- ❤️ Like artworks and add them to your favorites
- 🔒 Private pages for adding artworks, viewing personal gallery, and managing favorites (Firebase Auth)
- 🌗 Responsive UI with light/dark theme toggle and smooth animations

---

## 🛠 Tech Stack

- **Frontend:** React (Vite), Tailwind CSS  
- **Backend:** Express, MongoDB Atlas, Firebase Auth  
- **Deployment:** Vercel  
- **Libraries/Dependencies:** react-image-gallery, react-awesome-reveal, react-tooltip, sweetalert2  

---

## 🚀 Run Locally

1. Clone the repo: `git clone https://github.com/Fahim20223/artify.git`  
2. Install dependencies: `npm install`  
3. Start development server: `npm run dev`  
4. Open `http://localhost:5173` in your browser
